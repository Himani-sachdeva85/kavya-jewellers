# Purpose
Build a luxury jewellery website frontend (Kavya Jewellers) using HTML, CSS, and JavaScript with clean UI and dynamic product behavior.

# When to use
Use when creating a stylish, modern jewellery brand website with product browsing, cart system, and responsive layout.

# Features
- Multi-page architecture (Home, Collections, Bridal, About, Contact, Cart)
- Premium Luxury UI (Gold & Ivory aesthetic)
- 100 Dynamic Products with Infinite Scroll
- Cart & Wishlist persistence (LocalStorage)
- Responsive design for Mobile & Desktop
- SEO Optimized semantic HTML
